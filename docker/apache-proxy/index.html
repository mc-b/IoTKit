IoTKit USB Stick

Der IoTKit USB Stick fasst Server und Tools für die Kursreihe Internet der Dinge zusammen. Der IoTKit USB Stick baut dabei auf XUbuntu und Docker auf. Zusätzlich wird ein Account auf der mbed Developer Site und ein Freescale FRDM-K64F Board benötigt. Ein IoTKit SMD Shield ist von Vorteil, aber nicht Bedingung.
