IoTKit USB Stick

Der IoTKit USB Stick ergänzt die IoTKit Hardware mit Services & Frameworks und Tools. Der IoTKit USB Stick baut dabei auf XUbuntu und Docker auf. Zusätzlich wird ein Account auf der mbed Developer Site und ein Freescale FRDM-K64F Board benötigt. Ein IoTKit SMD Shield ist von Vorteil, aber nicht Bedingung.

Services & Frameworks

  • LAM Server (Linux, Apache, MySQL - HTTP, REST) - Logs, mbed, Downloads,
  • Smart Home (openHAB2 - HTTP, REST) - Logs, mbed, Java Binding
  • Heimautomatisierung (FHEM - HTTP) - Logs, mbed
  • MQTT Broker (Mosquitto) - Logs, mbed
  • LWM2M Server (Eclipse leshan - CoAP, REST) - Logs, mbed
  • IoT Data Plattform (ThingSpeak, - HTTP, REST) - Logs, mbed

Tools

  • ARM mbed Developer Site (FRMD K64F Board, IoTKit SMD Shield)
  • mbed IDE und Build Server (jenkins, yotta, gcc4arm) - Logs
  • Web-App IDE und Build Server (jenkins, Apache Cordova) - Logs
  • Java IDE und Build Server (jenkins, maven) - Logs
  • Linux Shell (wetty, web-terminal) - Logs
  • Netzwerk Übersicht
